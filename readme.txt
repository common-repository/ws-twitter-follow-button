== Description ==

WS Twitter Follow Button plugin provides a small button displayed on your websites to help users easily follow a Twitter account. 

<strong>Features</strong> 

* Easy to install and Configure
* User friendly and easy to use.
* Simply Add Follow Button
* Change language of follow button
* Easy to add shortcode anywhere in your wordpress site.

== Installation ==

1. Download [WS Twitter Follow Button](https://wordpress.org/plugins/ws-twitter-follow-button/) plugin.   
2. Upload the WS Twitter Follow Button plugin from your admin panel.    
3. Activate the WS Twitter Follow Button plugin. 
4. Add WS Twitter Follow Button widget to your sidebar.
